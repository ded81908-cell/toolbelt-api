QR codes, social/OG images, JSON↔YAML↔CSV conversion, hashing, encoding and generators — behind a single fast API. No AI bills, predictable pricing, live in minutes.

Convert

JSON ↔ YAML ↔ CSV.

POST /v1/hash

SHA/MD5/HMAC, base64, hex.

Generators

GET /v1/uuid

UUIDs, passwords, tokens, slugs.

curl -X POST https://your-host/v1/qr \
  -H "X-API-Key: your-key" \
  -H "Content-Type: application/json" \
  -d '{"text":"https://example.com","format":"png","size":512}' \
  --output qr.png

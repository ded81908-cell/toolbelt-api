Try it

One request, real output

# Generate a QR code as PNG
curl -X POST https://toolbelt-api-9oll.onrender.com/v1/qr \
  -H "X-RapidAPI-Key: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"text":"https://example.com","format":"png","size":512}' \
  --output qr.png

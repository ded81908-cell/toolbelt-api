### Cadence (avoid spam flags)
- **Day 1**: Qiita + Zenn (JP article), X(JP) thread
- **Day 2**: Dev.to (EN article), X(EN) thread
- **Day 3**: Reddit (one subreddit only), GitHub awesome-list PR
- **Day 4+**: Product Hunt launch (pick a Tue–Thu), Hacker News "Show HN"

> Don't cross-post the same link to many subreddits the same day — that triggers
> spam filters. One subreddit, genuine tone, reply to comments.
